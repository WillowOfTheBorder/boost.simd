template<typename T >
auto boost::simd::inrad ( T const &  x)

convert degree to radian.

Semantic:

For every parameter of floating type T

T r = inrad(x);

is similar to:

T r = Deginrad<T>()*x;
See also
indeg, Radindeg, Radindegr, Deginrad
Returns
a value of the same type as the parameter
